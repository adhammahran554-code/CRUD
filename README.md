# CRUDS Product Management System

A responsive web-based CRUDS application built using **HTML**, **CSS**, and **JavaScript**. The project allows users to efficiently manage products by creating, reading, updating, deleting, and searching product records. All data is stored locally using the browser's **Local Storage**, ensuring persistence after page refresh.

## Features

* Create new products
* Calculate total price automatically (Price + Taxes + Ads - Discount)
* Create multiple products at once using the Count field
* Update existing products
* Delete individual products
* Delete all products with one click
* Search products by Title or Category
* Light/Dark mode with saved user preference
* User welcome message stored in Local Storage
* Responsive and clean user interface
* Data persistence using Local Storage

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## How to Use

1. Enter the product information.
2. Click **Create** to save the product.
3. Use **Update** to edit an existing product.
4. Use **Delete** to remove a product.
5. Search by **Title** or **Category**.
6. Toggle between **Light** and **Dark** modes.
